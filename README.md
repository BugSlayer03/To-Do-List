# 📝 To-Do List App

A simple and efficient To-Do List web application that helps users manage daily tasks with ease. Tasks can be added, viewed, and deleted, with data stored locally in the browser for persistence.

## 🚀 Features

* ➕ Add new tasks with a due date
* 🗑️ Delete tasks instantly
* 💾 Data persistence using Local Storage
* 📅 Track tasks with associated dates
* 🎨 Clean and minimal user interface
* ⚡ Fast and lightweight (no external libraries)

## 🛠️ Tech Stack

* **HTML5**
* **CSS3 (Grid Layout)**
* **JavaScript (ES6)**
* **LocalStorage API**

## ⚙️ How It Works

1. Enter a task name in the input field
2. Select a due date
3. Click the **Add** button to store the task
4. Tasks are displayed in a structured list
5. Click **Delete** to remove a task
6. All tasks are saved in browser Local Storage

## ⚠️ Note

* Tasks are stored in the browser’s Local Storage, so clearing browser data will remove all tasks.
* This app does not use any backend or database.
